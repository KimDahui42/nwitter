## 09.23
로그인 기능 추가
절대 경로의 아쉬운 점, `npm install`로 설치한 패키지 이름이 파일 이름과 같으면 오류가 발생한다. 
* `firebase.js->fbase.js`
`jsconfig.json` src 폴더를 기준으로 폴더나 파일 경로를 적을 수 있어 import 문의 가독성이 좋아진다
